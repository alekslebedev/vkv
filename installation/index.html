vkv

Version v0.1.1

export/import secrets from Vaults KV2 engine

Installation

Using go

go install github.com/FalcoSuessgott/vkv@latest

From Sources

git clone https://github.com/FalcoSuessgott/vkv
cd vkv
go install

Packages

vkv has RPM, DEB packages and Windows Binaries in the release section.
